OVI uso de editores web online para GIT/GITHUB, HTML5 y CSS3.

El presente Objeto virtual de información, pretende establecer los principales conceptos necesarios para el manejo adecuado de un editor web entre distintos aplicativos webs online se determina lo básico par a su manejo y comprensión de tal manera que se note la importancia de su uso para cualquier desarrollador o programador web.

En este OVI, se muestran diversas formas sobre el concepto de la edición de texto para la web ya que permite identificar ideas y contextos que se permiten comprender algunas reglas generales del uso de las herramientas más utilizadas en la edición web, tales como la búsqueda de datos, el reemplazamiento de palabras, la eliminación de símbolos especiales, la comparación de textos, la eliminación de palabras repetidas, la búsqueda de un punto intermedio hacia abajo o hacia arriba, determinar si las palabras con mayúsculas o con acentos son iguales a las que están escritas igual pero sin acento y en minúscula.
