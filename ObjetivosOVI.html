OVI uso de editores web online para GIT/GITHUB, HTML5 y CSS3.

El presente Objeto virtual de información, pretende establecer los principales conceptos necesarios para el manejo adecuado de un editor web entre distintos aplicativos webs online se determina lo básico par a su manejo y comprensión de tal manera que se note la importancia de su uso para cualquier desarrollador o programador web.

  • Objetivo General
  • Objetivos Especificos
  • Participantes

Objetivo específico 1:

Utilizar elementos didácticos que aumenten el interés y la capacidad de compresión acerca de cómo utilizar los editores de texto online y su importancia a la hora de construir un desarrollo web.

Objetivo específico 2:

Crear un espacio que sirva como guía inicial y pueda ser utilizada para adquirir los conocimientos básicos de la búsqueda, edición, y reemplazo de los datos en un texto.

Objetivo específico 3:

Mostrar varias funcionalidades para la edición web, tales como palabras primitivas, colores aplicados a palabras clave, selección de editor según lenguaje trabajo, edición de múltiples tipos de extensiones de archivos, búsqueda ágil, reemplazado de datos en vertical, entre otros.
